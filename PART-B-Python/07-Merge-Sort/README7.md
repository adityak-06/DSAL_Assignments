# 📦 Merge Sort (Online Orders)

## 📌 Problem Statement

Sort online orders based on delivery time using Merge Sort.

## ⚙️ Features

* Stable sorting
* Divide and Conquer

## 🧠 Concepts Used

* Recursion
* Merge process

## ⏱ Complexity

* Time → O(n log n)
* Space → O(n)

## ▶️ Run

```
python merge_sort_orders.py
```
