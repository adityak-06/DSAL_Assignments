# 🌳 Binary Search Tree (BST)

## 📌 Problem Statement

Implement BST with advanced operations:

* Insert (with duplicates)
* Delete
* Search
* Traversals
* Depth calculation
* Mirror image
* Copy tree
* Parent & Leaf nodes
* Level-wise display

## ⚙️ Features

* Duplicate handling
* BFS traversal
* Tree transformations

## 🧠 Concepts Used

* Trees
* Recursion
* BFS (Queue)

## ⏱ Complexity

* Average → O(log n)
* Worst → O(n)

## ▶️ Run

```
python bst_operations.py
```
