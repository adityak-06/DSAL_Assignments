# 🎒 Fractional Knapsack

## 📌 Problem Statement

Maximize profit by selecting items (even partially) within weight limit.

## ⚙️ Features

* Greedy selection
* Profit/Weight ratio sorting

## 🧠 Concepts Used

* Greedy Algorithms

## ⏱ Complexity

* Time → O(n log n)

## ▶️ Run

```
python fractional_knapsack.py
```
