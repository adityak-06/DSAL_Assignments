# 🌐 Minimum Spanning Tree (Graph)

## 📌 Problem Statement

Represent a graph and find MST using:

* Kruskal’s Algorithm
* Prim’s Algorithm

## ⚙️ Features

* Edge sorting (Kruskal)
* Greedy approach (Prim)
* Graph representation

## 🧠 Concepts Used

* Graph Theory
* Greedy Algorithms
* Union-Find

## ⏱ Complexity

* Kruskal → O(E log E)
* Prim → O(V²)

## ▶️ Run

```
python kruskal.py
python prim.py
```
