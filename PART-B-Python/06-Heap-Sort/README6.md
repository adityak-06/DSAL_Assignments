# 🔢 Heap Sort

## 📌 Problem Statement

Sort an array using Heap Sort algorithm.

## ⚙️ Features

* Max Heap construction
* In-place sorting

## 🧠 Concepts Used

* Binary Heap
* Tree representation using arrays

## ⏱ Complexity

* Time → O(n log n)
* Space → O(1)

## ▶️ Run

```
python heap_sort.py
```
