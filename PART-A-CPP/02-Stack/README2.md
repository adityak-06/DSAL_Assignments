# 📚 Stack using Linked List

## 📌 Problem Statement

Implement a stack using linked list and use it for:

* Infix → Postfix conversion
* Infix → Prefix conversion

## ⚙️ Features

* Linked list-based stack
* Operator precedence handling
* Expression validation

## 🧠 Concepts Used

* Stack ADT
* Linked List
* Expression Parsing

## ⏱ Complexity

* Time → O(n)

## ▶️ Run

```
g++ infix_conversion.cpp
./a.out
```
