# 🔍 Searching & Sorting (Student Database)

## 📌 Problem Statement

Design a student database system using dynamic memory allocation that supports:

* Adding student records (ID, Name, CGPA)
* Searching (Linear & Binary)
* Sorting (Name & CGPA)

## ⚙️ Features

* Dynamic memory allocation
* Linear Search
* Binary Search
* Bubble Sort (Name)
* Selection Sort (CGPA, ID)

## 🧠 Concepts Used

* Arrays & Structures
* Pointers
* Sorting Algorithms
* Searching Algorithms

## ⏱ Complexity

* Linear Search → O(n)
* Binary Search → O(log n)
* Sorting → O(n²)

## ▶️ Run

```
g++ student_management.cpp
./a.out
```
