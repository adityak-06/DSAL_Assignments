# 🔄 Circular Queue (Array Implementation)

## 📌 Problem Statement

Implement a circular queue with fixed size and perform:

* Enqueue
* Dequeue
* Display

## ⚙️ Features

* Circular indexing
* Overflow & Underflow handling

## 🧠 Concepts Used

* Queue ADT
* Modulo arithmetic

## ⏱ Complexity

* Enqueue → O(1)
* Dequeue → O(1)

## ▶️ Run

```
g++ circular_queue.cpp
./a.out
```
